# BottomTabbar

### 一、项目介绍

Android 底部导航栏3种 UI 模板

如果动图加载失败，可以参考下面的静图

### 三、风格说明

1. 风格1是使用 Android 原生的 BottomNavigationView 控件
2. 风格2是使用的 RadioGroup + ViewPager 实现的一个效果
3. 风格3也是使用的 RadioGroup，只不过中间多了一个大按钮可以跳转到另一个页面
